**Course:** Code with Harry — The Ultimate Python Course

## About This Chapter
This chapter covers the fundamental building blocks of Python programming. Variables store data, data types define what kind of data is stored, and operators perform calculations and comparisons on that data.

## Key Concepts
- **Variables:** Named memory locations to store values
- **Data Types:** int, float, str, bool, None
- **Typecasting:** Converting between data types using int(), float(), str()
- **Input/Output:** Taking user input with input() and displaying output with print()
- **Operators:** Arithmetic (+, -, *, /, //, %, **), Comparison (==, >, <, >=, <=, !=), Logical (and, or, not), Assignment (=, +=, -=, *=, /=)
